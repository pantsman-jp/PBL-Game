# PBL-Game

[日本語](README.ja.md)

## Overview
This game was created to **introduce high school students
to the exciting fields of space engineering and space science**.

## Installation
```
git clone https://github.com/pantsman-jp/PBL-Game
```

## Usage
Require `Python3.13.9`, `Pygame2.6.1`.

```
pip install pygame
```

To run,
```
python -m src.main
```
