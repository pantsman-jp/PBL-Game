# PBL-Game

## 概要
このゲームは、**宇宙工学・宇宙理学の面白さを高校生に紹介する** というコンセプトで開発されています。

## 免責事項
本ゲームの利用に際しては [免責事項](docs/disclaimer.ja.md) をご確認ください。

## インストール方法
```
git clone https://github.com/pantsman-jp/PBL-Game
```

## 使い方
`Python3.13.9`, `Pygame2.6.1` が必要。

`Pygame` は以下でインストール。

```
pip install pygame
```

実行方法は以下。
```
python -m src.main
```
