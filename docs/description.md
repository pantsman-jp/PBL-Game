# 説明資料

## 概要
このゲームは、大学構内および日本各地を舞台に、RPG とビジュアルノベル要素を組み合わせた学習・探究型ゲームです。
プレイヤーは NPC との会話、クイズ、アイテム収集を通じてストーリーを進め、
各地域のテーマ（構造系・制御系・通信・燃料・発射など）に関連する知識を体験的に学ぶことができます。

## デモ版の特徴
- デモ版であり、本リリースではありません
- 操作や表示に不具合が生じる可能性があります

## ゲームの基本構成
### 1. RPG パート（マップ探索）
- `WASD` キーで移動、`Z` キーで NPC と会話
- タイルベースのマップで地形による移動制限を実装
- マップ間移動時にアイリスイン・アウトのアニメーション

### 2. ビジュアルノベルパート
- 特定 NPC やイベント地点でノベルパートに遷移
- 背景画像・立ち絵・テキストウィンドウでストーリー表示
- 選択肢は GUI ボタンで表示、マウス・キー操作対応

### 3. クイズ・報酬システム
- NPC 会話中にクイズが発生する場合あり
- 正誤判定に応じてアイテム獲得
- 同一アイテムの重複取得は防止

## ストーリー進行
### 開始地点
- 北九州
- 画面上部に次の目的地や NPC 座標を表示

### 現在の進行順
1. 九州工業大学で開始
2. 指定 NPC との会話・クイズ
3. 次のマップへの移動イベント

ノベルパートとマップ探索が交互に発生し、段階的にストーリーが展開します。

## マップ構成（デモ版）
- 福岡県（開始地点）
- 山口県
- 愛知県
- 神奈川県
- 茨城県

各マップに NPC を配置し、会話やイベントで次の地域へ移動する仕組み。

## 操作方法

| 操作 | 内容 |
| :---: | :---: |
| `W` / `A` / `S` / `D` | プレイヤー移動 |
| `Z` | 会話・選択決定 |
| 矢印キー | クイズ・選択肢の移動 |
| `M` | マップ表示切り替え |
| `I` | インベントリ開閉 |

マップ表示中は移動・会話操作は無効。

## 技術的特徴
### システム設計
- `Python` + `Pygame`
- タイルベース描画、カリング処理による描画負荷軽減
- データ駆動型設計（NPC、マップ、シナリオを JSON 管理）

### オーディオ
- マップごとに BGM 切り替え
- OS ごとにミキサー初期化制御、無音バグを回避

### リソース管理
- PyInstaller 環境下でも動作するパス解決ロジック
- 画像・音声・フォントを一元管理

## 教育的意義
- 実際の地域や施設を舞台にすることで、学習内容を体験的に理解
- クイズや選択肢による即時フィードバックで知識定着を促進
- RPG とノベルの融合により、楽しみながら探究型学習を実現
